# Parameters

- `RADI`: smoothing radius used on tonal rasters before edge detection.
- `GTHR`: edge sensitivity threshold used by the Canny-based path.
- `LTHR`: minimum extracted skeleton length in pixels.
- `FTHR`: polyline simplification tolerance in output map units.
- `ATHR`: maximum angular difference allowed when linking segments.
- `DTHR`: maximum endpoint distance in pixels allowed when linking segments.
